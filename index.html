General information

There is a git repository for DIKU Systems Seminar on Github. Minutes and other material such as slides will be collected there over time.

The papers which are presented are mainly selected from an initial reading list, and inspired by similar seminars at Brown, Cornell and Berkeley. Minutes from the meetings are collected in the minutes directory of the git repository.
